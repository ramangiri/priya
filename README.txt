How to make live with GitHub Pages:
1. Create a public repo in GitHub.
2. Upload all files inside this folder.
3. In GitHub repo -> Settings -> Pages.
4. Source: Deploy from a branch.
5. Branch: main and folder: /(root).
6. Save.
7. Open the site URL GitHub shows.
